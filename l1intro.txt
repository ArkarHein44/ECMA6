=> ES6 ( Variables are declared by : let/ const)

var 
    - Global and block scope (function , condition, loop)
    - Value can change
    - Var shouldn't be used in ES6

let 
    - Block scope   
    - Value can change

const 
    - Block scope
    - Value can't change
